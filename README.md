# pi-packages

Pi extension monorepo.

## Packages

| Package | Description |
| --- | --- |
| [pi-caveman](./pi-caveman) | Caveman mode — terse output, ~75% fewer tokens, full technical accuracy |
| [pi-simplify](./pi-simplify) | Review recently changed code for clarity, consistency, maintainability |
| [pi-station](./pi-station) | Station bar status extension for pi coding agent |

## Build

```bash
bash build.sh [package-name]
```

## Publish

```bash
bash publish.sh [package-name]
```
